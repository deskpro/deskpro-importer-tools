Settings
========

### Base fields

| Column name | Type   | Required | Description           |
| ------------|--------|----------|-----------------------|
| name        | string |  *       | DeskPRO setting name. |
| value       | string |  *       | Setting value.        |

### Available settings list

| Setting name  | Description               |
|---------------|---------------------------|
| core.site_url | Your company website url. |
